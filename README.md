🚀 https://book-tracker-ui.vercel.app

---

## ✨ Features

- 🔐 Authentication
- 📚 Search books
- ➕ Add books to your library
- ⭐ Rate books
- 📖 Books status
- 🗑 Remove books from library
- ⚡ Cached requests with TanStack Query
- 🏗 Feature-Sliced Design architecture

---

## 🛠 Tech Stack

### Core

- Next.js (App Router)
- React
- TypeScript

### State & Data

- TanStack Query
- NextAuth

### UI

- Tailwind CSS
- Shadcn UI
- Lucide React

### Architecture

- Feature-Sliced Design (FSD)

---

## 🚀 Getting Started

Install dependencies:

```bash
bun install
```

Run development server:

```bash
bun run dev
```

Open:

```txt
http://localhost:3000
```
